# svelte-leaflet

Svelte component for [leaflet](https://leafletjs.com/)

[Documentation & Demo](https://ngyewch.github.io/svelte-leaflet/)

[Sample project](https://github.com/ngyewch/svelte-leaflet-test)

## Component support

- [x] Map

UI layers:

- [x] Marker
- [x] Popup
- [x] Tooltip

Raster layers:

- [x] TileLayer
- [x] TileLayer.WMS
- [x] ImageOverlay
- [ ] VideoOverlay

Vector layers:

- [x] Polyline
- [x] Polygon
- [x] Rectangle
- [x] Circle
- [x] CircleMarker
- [ ] SVGOverlay

Other layers:

- [ ] LayerGroup
- [ ] FeatureGroup
- [x] GeoJSON

Basic types:

- [x] Icon
- [x] DivIcon

Controls:

- [ ] Zoom
- [ ] Attribution
- [ ] Layers
- [x] Scale
